# PerfectMatching
This is a simple script for counting perfect matching on given graph.  
Graph.rb provides some functions which generates graphs of specific type.

## PerfectMatching.rb
* PerfectMatching(graph)  

## Graph.rb
* AztecDiamond(n)  
* SquareLattice(n,m)
